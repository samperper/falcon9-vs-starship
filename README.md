# The SpaceX Fleet: A Financial Breakdown

Live URL: https://falcon9-vs-starship.vercel.app

The SpaceX Fleet: A Financial Breakdown is an interactive launch-economics model comparing Falcon 9, Falcon Heavy, and Starship costs, pricing, payload economics, customers, and cost-per-kilogram trajectories.

Author: Sam Perper

Methodology: The model is sourced from public filings, SpaceX pricing materials, government reports, and analyst research from Payload Research, Sacra, and related launch-market references; assumptions are adjustable through the interactive model so the economics can be tested under different scenarios.
